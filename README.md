# Bhavani_minor_project1
📊 GroupDNA – WhatsApp Chat Analyzer

📖 Overview

GroupDNA – WhatsApp Chat Analyzer is a Python-based Data Analytics project that analyzes an exported WhatsApp group chat and generates meaningful insights about communication patterns, participant activity, and group behavior.

The project was developed using only basic Python concepts such as file handling, lists, dictionaries, sets, loops, functions, string operations, and NumPy, following the project guidelines.

---

🎯 Objectives

- Analyze WhatsApp chat data.
- Generate useful chat statistics.
- Identify participant activity.
- Understand communication patterns.
- Apply Python programming concepts to a real-world dataset.

---

✨ Features

- 📩 Total Messages Count
- 👥 Total Participants
- 💬 Messages Sent by Each Participant
- 🏆 Most Active Member
- 📉 Least Active Member
- ✍ Average Message Length
- 🔤 Top 10 Most Common Words
- 🖼 Media Shared Count
- 🗑 Deleted Messages Count
- ⚙ System Messages Count
- 📅 Most Active Day
- 🕒 Most Active Hour
- 🔥 Activity Heatmap using NumPy
- 🎭 Personality Archetype Detection
- 📄 Final Chat Analysis Report
